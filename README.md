# Smart Probashi – স্মার্ট প্রবাসী

Mobile-first PWA for Bangladeshi expats to track income, expenses, and goals.

## Features
- Income Tracking (আয়)  
- Expense Tracking (খরচ)  
- Goal Management (লক্ষ্য)  
- Profile Management (প্রোফাইল)  
- Dashboard (ড্যাশবোর্ড summary)

## Tech Stack
- Frontend: Next.js + Tailwind CSS  
- Backend: Firebase Auth + Firestore + Hosting  
- PWA Support

## How to Run Locally
1. Clone Repo: `git clone https://github.com/<your-username>/smart-probashi.git`
2. `cd smart-probashi/frontend/nextjs-tailwind`
3. Install dependencies: `npm install`
4. Run dev server: `npm run dev`
